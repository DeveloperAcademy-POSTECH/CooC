# :iphone: A&A

"What should I have for lunch today?" Ask questions whenever you're hesitated with concerns in daily life.


## :pushpin: Features

- See popular/recent topics
- Search topics
- Upload topic
- Real-time voting
- Edit profile
- View the list of topics users have posted and responded to
